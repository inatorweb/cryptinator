## Core Encryption

### libsodium
**License:** ISC License  
**Copyright:** Copyright (c) 2013-2024 Frank Denis and contributors  
**Repository:** https://github.com/jedisct1/libsodium  
**Website:** https://libsodium.org  

Industry-standard cryptographic library providing XChaCha20-Poly1305 encryption and Argon2id key derivation. Used by Signal, WhatsApp, GitHub, Cloudflare, and thousands of production applications worldwide.

### sodium_libs
**License:** BSD 3-Clause License  
**Copyright:** Copyright (c) 2020-2024 Samuel Lucas  
**Repository:** https://github.com/nickvdyck/sodium_libs  

Precompiled libsodium binaries for Flutter platforms (Windows, macOS, Linux, Android, iOS).

### flutter_sodium
**License:** BSD 3-Clause License  
**Copyright:** Copyright (c) 2020-2024 Samuel Lucas  
**Repository:** https://github.com/nickvdyck/flutter_sodium  

Dart API wrapper for libsodium, providing easy-to-use cryptographic functions.
